# AI Presentation Performance Analyzer

An AI-powered system that evaluates presentation performance using
Computer Vision, Speech Processing, and Natural Language Processing.

## Features

• Facial emotion recognition
• Speech delivery analysis
• Language quality evaluation
• AI confidence score
• Emotion distribution visualization
• Speech analytics dashboard

## AI Modules

Computer Vision
- CNN-based facial emotion recognition

Speech Processing
- Whisper speech transcription
- Words-per-minute analysis

Natural Language Processing
- Vocabulary richness
- Filler word detection
- Sentiment analysis

## Tech Stack

Frontend
React.js
Chart.js

Backend
FastAPI
PyTorch
OpenAI Whisper
NLTK

Video + Audio Input
